# DDA-Based Instrumentation Amplifier (180nm CMOS)

## Overview
This project presents the design and simulation of a Differential Difference Amplifier (DDA) based Instrumentation Amplifier using 180nm CMOS technology. The design is targeted for low-amplitude biomedical signals such as ECG.

## Key Highlights
- High Gain: ~48 dB
- High CMRR: ~109 dB
- Low Power: ~3.28 µW
- Bandwidth: ~150 Hz

## Architecture
The amplifier consists of:
- Differential Difference Amplifier (DDA)
- Folded Cascode Gain Stage
- Common Mode Feedback (CMFB)
- Differential to Single-Ended Output Stage

## Tools Used
- Cadence Virtuoso
- gpdk180 CMOS Technology

## Applications
- ECG Signal Acquisition
- Biomedical Front-End Circuits
- Low Power Analog Systems
